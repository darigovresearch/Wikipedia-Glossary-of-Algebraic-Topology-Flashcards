"Introduction (Wikipedia Glossary of Algebraic Topology)"¬"
<p>This deck has taken definitions outlined in <a href=""https://en.wikipedia.org/wiki/Glossary_of_algebraic_topology"">Wikipedia's Glossary of algebraic topology</a> and put them into a form which can be easily learnt/revised using <a href=""https://apps.ankiweb.net/"">Anki</a> a cross platform app specifically designed for long term knowledge retention.</p>
<h2>Notes</h2>
<p>Please note the modifications which have been made &amp; where you can find updates.</p>
<ol align=""left"">
<li>The front of every card has ""(Wikipedia Glossary of Algebraic Topology)"" appended to the end so that if you have any other words in your collection, the Wikipedia definition will still be added when importing it.</li>
<li>Any updates or corrections to the deck will be available at <a href=""https://github.com/darigovresearch/Wikipedia-Glossary-of-Algebraic-Topology-Flashcards"">https://github.com/darigovresearch/Wikipedia-Glossary-of-Algebraic-Topology-Flashcards</a> so do return periodically to check if you have the latest version.</li>
</ol>
<p>Feel free to share the deck and give the repository a star so more people are likely to see this work and can get the most out of it.</p>

"chain homotopy equivalence (Wikipedia Glossary of Algebraic Topology)"¬"
A chain map that is an isomorphism up to chain homotopy; that is, if <i>ƒ</i>:<i>C</i>→<i>D</i> is a chain map, then it is a chain homotopy equivalence if there is a chain map <i>g</i>:<i>D</i>→<i>C</i> such that <i>g</i>ƒ and ƒ<i>g</i> are chain homotopic to the identity homomorphisms on <i>C</i> and <i>D</i>, respectively.

"covering (Wikipedia Glossary of Algebraic Topology)"¬"
<dd class=""glossary"">1.&#160;&#160;A map <i>p</i>: <i>Y</i> → <i>X</i> is a <a href=""https://en.wikipedia.org/wiki/Covering_space"" title=""Covering space"">covering</a> or a covering map if each point of <i>x</i> has a neighborhood <i>N</i> that is <a href=""https://en.wikipedia.org/w/index.php?title=Evenly_covered&amp;action=edit&amp;redlink=1"" class=""new"" title=""Evenly covered (page does not exist)"">evenly covered</a> by <i>p</i>; this means that the pre-image of <i>N</i> is a disjoint union of open sets, each of which maps to <i>N</i> homeomorphically.</dd>
<dd class=""glossary"">2.&#160;&#160;It is <i>n</i>-sheeted if each fiber <i>p</i><sup>−1</sup>(<i>x</i>) has exactly <i>n</i> elements.</dd>
<dd class=""glossary"">3.&#160;&#160;It is <a href=""https://en.wikipedia.org/wiki/Universal_covering"" class=""mw-redirect"" title=""Universal covering"">universal</a> if <i>Y</i> is simply connected.</dd>
<dd class=""glossary"">4.&#160;&#160;A morphism of a covering is a map over <i>X</i>. In particular, an automorphism of a covering <i>p</i>:<i>Y</i>→<i>X</i> (also called a <a href=""https://en.wikipedia.org/wiki/Deck_transformation"" class=""mw-redirect"" title=""Deck transformation"">deck transformation</a>) is a map <i>Y</i>→<i>Y</i> over <i>X</i> that has inverse; i.e., a homeomorphism over <i>X</i>.</dd>
<dd class=""glossary"">5.&#160;&#160;A <a href=""https://en.wikipedia.org/w/index.php?title=G-covering&amp;action=edit&amp;redlink=1"" class=""new"" title=""G-covering (page does not exist)""><i>G</i>-covering</a> is a covering arising from a <a href=""https://en.wikipedia.org/wiki/Group_action_(mathematics)"" class=""mw-redirect"" title=""Group action (mathematics)"">group action</a> on a space <i>X</i> by a group <i>G</i>, the covering map being the quotient map from <i>X</i> to the <a href=""https://en.wikipedia.org/wiki/Orbit_space"" class=""mw-redirect"" title=""Orbit space"">orbit space</a> <i>X/G</i>. "fiber-homotopy equivalence (Wikipedia Glossary of Algebraic Topology)"¬"
Given <i>D</i>→<i>B</i>, <i>E</i>→<i>B</i>, a map ƒ:<i>D</i>→<i>E</i> over <i>B</i> is a <a href=""https://en.wikipedia.org/wiki/Fiber-homotopy_equivalence"" title=""Fiber-homotopy equivalence"">fiber-homotopy equivalence</a> if it is invertible up to homotopy over <i>B</i>. The basic fact is that if <i>D</i>→<i>B</i>, <i>E</i>→<i>B</i> are fibrations, then a homotopy equivalence from <i>D</i> to <i>E</i> is a fiber-homotopy equivalence.

"homotopy category (Wikipedia Glossary of Algebraic Topology)"¬"
Let <i>C</i> be a subcategory of the category of all spaces. Then the <a href=""https://en.wikipedia.org/wiki/Homotopy_category"" title=""Homotopy category"">homotopy category</a> of <i>C</i> is the category whose class of objects is the same as the class of objects of <i>C</i> but the set of morphisms from an object <i>x</i> to an object <i>y</i> is the set of the homotopy classes of morphisms from <i>x</i> to <i>y</i> in <i>C</i>. For example, a map is a homotopy equivalence if and only if it is an isomorphism in the homotopy category.

"homotopy equivalence (Wikipedia Glossary of Algebraic Topology)"¬"
<dd class=""glossary"">1.&#160;&#160;A map ƒ:<i>X</i>→<i>Y</i> is a <a href=""https://en.wikipedia.org/wiki/Homotopy_equivalence"" class=""mw-redirect"" title=""Homotopy equivalence"">homotopy equivalence</a> if it is invertible up to homotopy; that is, there exists a map g: <i>Y</i>→<i>X</i> such that <i>g</i> ∘ ƒ is homotopic to th identity map on <i>X</i> and ƒ ∘ <i>g</i> is homotopic to the identity map on <i>Y</i>.</dd>
<dd class=""glossary"">2.&#160;&#160;Two spaces are said to be homotopy equivalent if there is a homotopy equivalence between the two. For example, by definition, a space is contractible if it is homotopy equivalent to a <a href=""https://en.wikipedia.org/w/index.php?title=Point_space&amp;action=edit&amp;redlink=1"" class=""new"" title=""Point space (page does not exist)"">point space</a>.</dd>

<Lazard ring (Wikipedia Glossary of Algebraic Topology)"¬"
The <a href=""https://en.wikipedia.org/wiki/Lazard_ring"" class=""mw-redirect"" title=""Lazard ring"">Lazard ring</a> <i>L</i> is the (huge) commutative ring together with the <a href=""https://en.wikipedia.org/wiki/Formal_group_law"" title=""Formal group law"">formal group law</a> ƒ that is universal among all the formal group laws in the sense that any formal group law <i>g</i> over a commutative ring <i>R</i> is obtained via a ring homomorphism <i>L</i> → <i>R</i> mapping ƒ to <i>g</i>. According to Quillen's theorem, it is also the coefficient ring of the <a href=""https://en.wikipedia.org/wiki/Complex_bordism"" class=""mw-redirect"" title=""Complex bordism"">complex bordism</a> MU. The <a href=""https://en.wikipedia.org/wiki/Spectrum_of_a_ring"" title=""Spectrum of a ring"">Spec</a> of <i>L</i> is called the <a href=""https://en.wikipedia.org/wiki/Moduli_space_of_formal_group_laws"" class=""mw-redirect"" title=""Moduli space of formal group laws"">moduli space of formal group laws</a>.

"simple-homotopy equivalence (Wikipedia Glossary of Algebraic Topology)"¬"
A map ƒ:<i>X</i>→<i>Y</i> between finite simplicial complexes (e.g., manifolds) is a <a href=""https://en.wikipedia.org/wiki/Simple-homotopy_equivalence"" title=""Simple-homotopy equivalence"">simple-homotopy equivalence</a> if it is homotopic to a composition of finitely many <a href=""https://en.wikipedia.org/w/index.php?title=Elementary_expansion&amp;action=edit&amp;redlink=1"" class=""new"" title=""Elementary expansion (page does not exist)"">elementary expansions</a> and <a href=""https://en.wikipedia.org/w/index.php?title=Elementary_collapse&amp;action=edit&amp;redlink=1"" class=""new"" title=""Elementary collapse (page does not exist)"">elementary collapses</a>. A homotopy equivalence is a simple-homotopy equivalence if and only if its <a href=""https://en.wikipedia.org/wiki/Whitehead_torsion"" title=""Whitehead torsion"">Whitehead torsion</a> vanishes.
